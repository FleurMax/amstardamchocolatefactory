# Project Summary: The Amsterdam Chocolate Factory Website

I have successfully created a high-end React-based website for **The Amsterdam Chocolate Factory**, following the specific "Steampunk-Meets-Willy-Wonka" aesthetic from your reference images.

## 🎨 Design Philosophy
The website combines the luxury of artisanal Belgian chocolate with the whimsical, industrial charm of 17th-century Amsterdam architecture.

- **Visual Palette**: Deep cocoa browns, burnished golds, and copper accents.
- **Maximalist Styling**: I moved away from minimalist glassmorphism toward a more "textured" look featuring:
    - **Wood-carved Textures**: Used for the header and footer to ground the design.
    - **Ornate Gold Frames**: In the "Experience Trio" section, each image is framed in double-lined gold borders, mimicking a gallery or an antique shop.
    - **Parallax Magic**: The "Golden Ticket" narrative section features a parallax-scrolling "River of Chocolate" background to create depth and wonder.
- **Typography**: I used **Playfair Display** (Serif) for headings to evoke a timeless, premium feel, and **Outfit** (Sans-serif) for readable, modern body text.

## 🛠️ Technical Implementation
The project is built using a modern React stack for performance and smooth animations:
- **Framework**: Vite + React
- **Animations**: [Framer Motion](https://www.framer.com/motion/) triggers smooth "reveal" effects as you scroll.
- **Icons**: [Lucide React](https://lucide.dev/) for crisp, stylized interface elements.
- **Responsive Design**: The layout automatically adjusts for mobile, tablet, and desktop viewing.
